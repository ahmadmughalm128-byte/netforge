# NetForge — Network Tools & Cybersecurity Learning Hub

&gt; **The ultimate platform for network engineers, cybersecurity students, and IT professionals.**

## Features

### Free Tools
- **Subnet Calculator** — Calculate subnets, broadcast addresses, usable ranges
- **CIDR Converter** — Convert between CIDR notation and IP ranges
- **Port Scanner/Lookup** — Lookup port numbers, services, and protocols
- **IP Classifier** — Classify IPs as Private, Public, Loopback, Multicast
- **Password Generator** — Generate secure passwords with entropy calculation

### Premium Tools (Pro)
- **MAC Address Lookup** — Identify vendor from OUI
- **Route Summarization** — Supernet calculation and aggregation
- **ACL Generator** — Generate Cisco ACLs and iptables rules

### Learning Hub
- **Network Fundamentals** — OSI model, TCP/IP, subnetting (Free)
- **Security Essentials** — Firewalls, ACLs, encryption, VPNs (Free)
- **CompTIA Network+ Prep** — 500+ exam questions with timed simulations (Pro)

### Quiz System
- Timed quizzes with countdown timer
- Instant feedback with detailed explanations
- Score tracking and performance analytics
